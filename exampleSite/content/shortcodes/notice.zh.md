通告简码提供4种样式，帮助优化页面结构。

### Note 注意

```
{{%/* notice note */%}}
A notice disclaimer
{{%/* /notice */%}}
```

效果如下：

{{% notice note %}}
A notice disclaimer
{{% /notice %}}
