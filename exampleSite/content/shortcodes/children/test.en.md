# Qué es Markdown

Markdown es un lenguaje de marcado ligero que puedes usar para añadir formato a texto plano.

## Elementos de bloque

### Párrafos y saltos de línea

El texto se convierte en párrafos separando las líneas con un espacio en blanco.

### Encabezados

# Encabezado 1
## Encabezado 2
### Encabezado 3

### Citas

> Esto es una cita.

### Listas

- Elemento 1
- Elemento 2
  - Subelemento 1
  - Subelemento 2
